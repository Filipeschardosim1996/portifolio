Lista de tags do HTML

  • < !DOCTYPE html > => Define o Tipo (html5)
  • < html > => Raiz do documento
  • < head >=> INformações da pagina(nao visiveis)
  • < body >=> Conteudo visivel da pagina
  • < header >=> Cabeçalho da pagina/seção
  • < main >=> Conteudo principal
  • < footer >=> Rodapé
  • < Section >=> Seção de conteudo
  • < article >=> Bloco independente(artigo)
  • < aside >=> Conteudo lateral/extra
  • < nav >=> Menu de navegação
  • < div >=> Divisão generica
  • < span >=> Trecho pequeno em linha
  • < h1_> < h6 > => Titulos de diferentes niveis
  • < p >=> Paragrafo
  • < br >=> Quebra de linha
  • < hr >=> Linha divisoria
  • < strong >=> Texto importante(negrito sermantico)
  • < em >=> Enfase (italico semantico)
  • < b >=> Negrito (Visual)
  • < i >=> Italico (Visual)
  • < u >=> Sublinhado
  • < mark >=> Texto marcado
  • < small >=> Texto menor
  • < sup / < sub >=> Sobrescrito / subscrito
  • < ul >=> Lista não ordenada
  • < ol >=> Lista ordenada
  • < li >=> Item de lista
  • < di >=> Lista de definições
  • < dt >=> Termo defino
  • < dd >=> Descrição de termo
  • < a >=> link
  • < img >=> imagem
  • < video >=> video
  • < audio >=> audio
  • < iframe >=> Conteudo externo embutido
  • < form >=> Formulario
  • < input >=> Campo de entrada
  • < testarea >=> Area de texto
  • < button >=> botão
  • < select > < option >=> Lista suspensa
  • < label >=> Rotulo de campo
  • < table >=> Tabela
  • < tr >=> Linha da tabela
  • < td >=> celular comum
  • < th >=> Cabeçalho da tabela
  • < thead > / / < tfoot >=> Estrutura tabela
